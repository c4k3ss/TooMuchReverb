## Key Features

* Reverb
* Downsampling
* Delay
* Uhh... knobs?
* It's free!

## How To Use

Run the [installer](https://github.com/c4k3ss/TooMuchReverb/releases), open your DAW, tweak the knobs... have fun!

> **Note**
> Be sure to download the installer for your OS (Windows/macOS)
